# Movie Timecode

Це [розширення](https://chromewebstore.google.com/detail/movie-timecode/oicfghfgplgplodmidellkbfoachacjb) допоможе вам автоматично приховувати небажані сцени під час трансляцій фільмів на Twitch, щоб уникнути порушень правил платформи *(але це не точно* 😅 *)*.

**Наразі розширення працює лише на сайтах uakino.best та uaserials.com.**

## Як працює цензурування?

1. Встановіть [розширення](https://chromewebstore.google.com/detail/movie-timecode/oicfghfgplgplodmidellkbfoachacjb).
2. Перейдіть на сайт UAKino і виберіть потрібний фільм.
3. Під відеоплеєром мають з'явитися нові кнопки управління. Натисніть кнопку пошук таймкодів, щоб знайти доступні таймкоди.
4. Підтвердіть вибір, і розширення автоматично активує цензурування зазначених сцен.

## Можливості розширення

- Можливість додавання власних таймкодів.
- Приховання небажаних сцен у фільмах по заданих таймкодах.
- Підключення до OBS або Streamlabs, щоб змінювати сцени під час цензурування.
- Можливість поверхневої перевірки фільму (рік, студії, дистриб'ютори, бани, вміст тощо). *Слід зазначити, що фільми в принципі не можна переглядати на Twitch.*
- Чат-бот, який дозволяє вашим глядачам користуватися командами в чаті, щоб дізнатися назву фільму, поточний час перегляду та виконувати інші корисні дії.
